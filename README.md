# 🧠 Neural-Anchor OS

The structural blueprint and "Operating System" for the Neural-Anchor knowledge system. 

## ⚓️ Concept
Neural-Anchor is designed for those who experience cognitive overload. It transforms raw information into structural assets, acting as a "cognitive proxy" that sustains thinking when the biological brain is exhausted.

## 🏗️ Architecture
- `prompts/`: The "Drivers" that power the knowledge extraction.
- `templates/`: The "Standardized Formats" for knowledge assets.
- `vault-template/`: The layout for the Obsidian vault.

## 🚀 Getting Started
1. Create a private repository (`neural-anchor-sanctuary`).
2. Use this repo as a template to initialize your vault.
3. Follow the `prompts/` to process your captured articles.
